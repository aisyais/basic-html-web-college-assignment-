Pengertian HTML

HTML adalah sebuah bahasa markup yang berfungsi untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser. Bentuk HTML berupa kode-kode yang menginstruksikan browser untuk menghasilkan tampilan seperti yang kita inginkan. Saat ini, HTML merupakan standar internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Ceb Consortium (W3C)

Sejarah HTML

  • HTML 2.0 : (RFC1866) disetujui sebagai standard pada 22 september 1995
  • HTML 3.2 : 14 Januari 1996
  • HTML 4.0 : 18 Desember 1997.
  • ISO/IEC 15445:2000 "ISO HTML" yang berdasar pada HTML 4.01 Strict (15 Mei 2000)
  • XHTML 1.0 : Gabungan HTML dan XML (24 Januari 200).
  • XHTML 1.1 : 16 Februari 2007.

Struktur HTML

  • <html></html>: document Information.
  • <head></head>: document header.
  • <body></body>: document body.
